# Online Shopping Store with Cart System

## Project Description

This project is a web-based online shopping store that includes a fully functional cart system. The application allows users to browse products, register and log in, add items to their personal cart, and manage their purchases.

The system also tracks user cart history and calculates the total price of selected items in real time.

## Features

### Cart System

* Add items to the cart
* Remove items from the cart
* Increase or decrease item quantity
* Automatic calculation of total price
* Display individual item prices
* Display total purchase amount
* Cart records include customer name, date, and time
* Clear cart functionality
* Checkout cart functionality
* Cart history showing previous purchases

### User Features

* User registration and login
* Browse products by category
* View additional pages such as:

  * Home
  * More About Us
  * Customer Support
  * Privacy Policy
  * Order History

Note: Users must be logged in before adding items to their cart.
